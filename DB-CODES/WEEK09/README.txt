# Week 9 - Sequelize Examples

Folders:
- ModelQueryingFinders: uses ORM methods (findAll, create, findByPk)
- ValidationConstraints: adds validations and constraints to models
- RawQueries: demonstrates raw SQL queries with sequelize.query()

How to run:
1. Copy `.env` with DB credentials into this folder.
2. Install packages: `npm install`
3. Run demos:
   - `node ModelQueryingFinders/index.js`
   - `node ValidationConstraints/index.js`
   - `node RawQueries/index.js`
